## 🚀 Project Overview

The **Inventory Management System** is a backend application built using **Spring Boot** to manage products, suppliers, and stock levels efficiently.  

It simulates a **real-world warehouse system**, enabling businesses to track inventory, manage stock flow, and maintain data consistency.

---

## 🎯 Key Features

- 📦 Product Management (Add, Update, Delete, View)
- 📊 Real-time Stock Tracking  
- 🔄 Stock In / Stock Out functionality  
- 🚚 Supplier Management  
- 🔍 Search & Filter Products  
- ⚠️ Low Stock Alerts  
- 🧱 Layered Architecture (Controller → Service → Repository)
